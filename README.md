### How to install:

  * **GNU/Linux instructions:**
  
    * Open the terminal and type the following command:
    
      ```shell
      $ sudo tar jxvf ffmpeg-4.4.1-linux-gnu-[arch]-build1.tar.bz2 -C /
      ````
